Eva, assistente virtual de agendamentos

  • Atendimento via Telegram
  • Gerenciamento de horários
  • Armazenamento de dados

EVA é um bot criado para auxiliar em serviços de agendamento virtual para que seus clientes possam marcar ou desmarcar horários sem sair de casa.

DialogFlow

JavaScript

Google Sheets

EVA é capaz de cadastrar clientes e realizar agendamentos atrelando as informações ao Google Sheets, onde é possível gerar relatório dos dados coletados.
